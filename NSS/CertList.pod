=head1 NAME

NSS::CertList - NSS Certificate lists

=head1 SYNOPSIS

  use 5.10.1;
  use Perl6::Slurp;

  use NSS;

  # create a certificate list
  my $certList = NSS::CertList->new();

  # add a NSS::Certificate to the list
  $certList->add($certificate);

  # verify with trusted certificate list
  my $res = $cert->verify_pkix(time, NSS::certUsageSSLServer, $certList);

  # get NSS::Certificate objects in lst
  my @certs = $certList->dump();


=head1 ABSRACT

Perl interface to NSS Certificate Lists

=head1 Description

At the moment certificate lists are mostly needed when you want to provide
a list of trust anchors to C<NSS::Certificate::verify_pkix>. They are also 
returned by a few functions.

=head1 FUNCTIONS

=over 4

=item B<new>

Create a new, empty C<CSS::CertList>

=item B<add ( CERTIFICATE )>

Add a C<NSS::Certificate> to the certificate list

=item B<dump>

Returns all C<NSS::Certificate>s that are in the certificate list as an array.

=back
